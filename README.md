
# Efficient data descriptors for identification of welding flaws

Does what it says on the tin!

Learns an effective PDE that describes collective data through a combination of spline optimisation, PDE learning, and subsequent re-projection capturing low-dimensional features within simulations that then act as a filter on real-data.
